# صفحات اللاندينج - Branch Hub

تم إنشاء صفحات لاندينج مختلفة لمساحة العمل Branch Hub، كل منها مصممة لتناسب احتياجات مختلفة:

## الصفحات المتاحة

### 1. صفحة Branch Hub الرئيسية (الصفحة الرئيسية الجديدة)
- **الرابط**: `/` (الصفحة الرئيسية)
- **الملف**: `resources/views/branchhub-landing.blade.php`
- **الوصف**: صفحة لاندينج مخصصة لـ Branch Hub مع تفاصيل مساحة العمل والحجز
- **المميزات**:
  - استخدام اللوجو الخاص بـ Branch Hub
  - ألوان مستوحاة من اللوجو (أسود، أحمر، أبيض)
  - تفاصيل عن مساحة العمل في قطاع غزة
  - خطط الحجز (يومي، أسبوعي، شهري)
  - استهداف الطلاب ورواد الأعمال والفريلانسرز
  - معلومات الموقع في فلسطين

### 2. الصفحة العصرية (Landing Page)
- **الرابط**: `/landing`
- **الملف**: `resources/views/landing.blade.php`
- **الوصف**: صفحة لاندينج عصرية مع تصميم متطور وتفاعلي
- **المميزات**:
  - تصميم متدرج جذاب
  - رسوم متحركة (AOS)
  - قسم المميزات مع بطاقات تفاعلية
  - قسم الإحصائيات
  - قسم الدعوة للعمل (CTA)
  - تصميم متجاوب (Responsive)

### 2. الصفحة المتطورة (Advanced Landing)
- **الرابط**: `/landing-advanced`
- **الملف**: `resources/views/landing-advanced.blade.php`
- **الوصف**: صفحة لاندينج متطورة مع مميزات إضافية
- **المميزات**:
  - شريط تنقل ثابت مع تأثيرات التمرير
  - تصميم أكثر تفصيلاً
  - قسم آراء العملاء (Testimonials)
  - رسوم متحركة متقدمة
  - عداد الإحصائيات المتحرك
  - تأثيرات بصرية متطورة

### 3. الصفحة البسيطة (Simple Landing)
- **الرابط**: `/landing-simple`
- **الملف**: `resources/views/landing-simple.blade.php`
- **الوصف**: صفحة لاندينج بسيطة وخفيفة
- **المميزات**:
  - تصميم نظيف وبسيط
  - تحميل سريع
  - محتوى مختصر
  - مناسب للأجهزة الضعيفة
  - تصميم متجاوب

### 4. الصفحة التقليدية (Welcome Page)
- **الرابط**: `/welcome`
- **الملف**: `resources/views/welcome.blade.php`
- **الوصف**: الصفحة التقليدية الموجودة مسبقاً

## كيفية التبديل بين الصفحات

### تغيير الصفحة الرئيسية
لتغيير الصفحة الرئيسية، قم بتعديل ملف `routes/web.php`:

```php
// للصفحة العصرية
Route::get('/', function () {
    if (Auth::check()) {
        return redirect()->route('dashboard');
    }
    return view('landing');
})->name('landing');

// للصفحة المتطورة
Route::get('/', function () {
    if (Auth::check()) {
        return redirect()->route('dashboard');
    }
    return view('landing-advanced');
})->name('landing');

// للصفحة البسيطة
Route::get('/', function () {
    if (Auth::check()) {
        return redirect()->route('dashboard');
    }
    return view('landing-simple');
})->name('landing');
```

## المميزات المشتركة

جميع الصفحات تحتوي على:

- **التصميم المتجاوب**: تعمل بشكل مثالي على جميع الأجهزة
- **خط Almarai**: خط عربي جميل ومقروء
- **أيقونات Bootstrap**: أيقونات متسقة وجميلة
- **ألوان متناسقة**: نظام ألوان موحد
- **روابط تسجيل الدخول**: توجيه المستخدمين للدخول للنظام

## التخصيص

يمكنك تخصيص أي من الصفحات عبر:

1. **تعديل الألوان**: في قسم `:root` في CSS
2. **تغيير المحتوى**: تعديل النصوص في HTML
3. **إضافة مميزات**: إضافة أقسام جديدة
4. **تعديل التصميم**: تغيير CSS حسب الحاجة

## الأداء

- **الصفحة البسيطة**: الأسرع في التحميل
- **الصفحة العصرية**: متوازنة بين الجمال والأداء
- **الصفحة المتطورة**: الأكثر تفاعلاً مع تحميل أبطأ قليلاً

## المتطلبات

جميع الصفحات تستخدم:
- Bootstrap 5.3.0
- Bootstrap Icons
- Google Fonts (Almarai)
- AOS Animation (للصفحات المتطورة)

## الدعم

لأي استفسارات أو تعديلات، يمكنك:
1. تعديل الملفات مباشرة
2. إضافة صفحات جديدة بنفس النمط
3. تخصيص التصميم حسب احتياجاتك

## تفاصيل صفحة Branch Hub

### المحتوى الرئيسي:
1. **قسم Hero**: عنوان جذاب مع اللوجو ووصف مساحة العمل
2. **قسم "عن Branch Hub"**: تفاصيل عن المساحة والمميزات
3. **قسم خطط الحجز**: 
   - حجز يومي: 25₪
   - حجز أسبوعي: 150₪ (مميز)
   - حجز شهري: 500₪
4. **قسم "مناسب لجميع الفئات"**: الطلاب، رواد الأعمال، الفريلانسرز
5. **قسم الموقع**: معلومات عن موقع Branch Hub في قطاع غزة
6. **قسم الدعوة للعمل**: حث المستخدمين على الحجز

### الألوان المستخدمة:
- **الأسود**: `#1a1a1a` (من اللوجو)
- **الأحمر**: `#dc2626` (من اللوجو)
- **الأبيض**: `#ffffff` (من اللوجو)
- **ألوان مساعدة**: رمادي فاتح، أخضر للنجاح

### الروابط المتاحة:
- `/` - صفحة Branch Hub الرئيسية (الجديدة)
- `/branchhub` - صفحة Branch Hub (رابط بديل)
- `/landing` - الصفحة العصرية
- `/landing-advanced` - الصفحة المتطورة
- `/landing-simple` - الصفحة البسيطة
- `/welcome` - الصفحة التقليدية
